# python-game

A linux terminal game where two players (1) and (2) use the keyboard to race to the goal ($) and navigate around obstacles (#)

Player one uses the 'w,a,s,z' keys.
Player two uses the 'i,j,k,m' keys.
First one to the '$' wins.
